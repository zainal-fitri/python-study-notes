2. Variables, Input & Casting

Ex 2: The Receipt Calculator

1. Create a variable price = "10.50" (String).
2. Create a variable quantity = 2 (Integer).
3. Calculate the total cost (Convert types where needed!).

Show Solution
price = "10.50"
qty = 2

# Must cast string to float for math
total = float(price) * qty
print(total) # 21.0

name = input("Name: ")

3. String Mechanics

Ex 4: The Email Extractor

Given email = " USER@Example.com ":
1. Remove extra spaces (strip).
2. Convert to lowercase.
3. Extract and print just the domain (slice from the '@' symbol to the end).

Show Solution
raw = " USER@Example.com "
clean = raw.strip().lower()
print(clean) # "user@example.com"

# Find position of @
at_index = clean.find("@")
domain = clean[at_index+1:] # Slice from after @ to end
print(domain) # "example.com"

5. Data Structures Deep Dive

Ex 7: List Queue

Start with queue = ["John", "Amy"].
1. Add "Bob" to end.
2. Remove "John" from start.
3. Print who is next (index 0).

Solution
q = ["John", "Amy"]
q.append("Bob")
q.pop(0) # Remove John
print(q[0]) # Amy

Ex 8: Deduping

You have a list with duplicates: [1, 2, 2, 3, 3, 3]. Use a Set to remove duplicates and print the unique count.

Solution

print(len(unique))

Ex 11: Sum of Evens (Loop)

Use a for loop with range() to iterate from 0 to 10. Add only the even numbers to a total variable.

Show Solution

for i in range(11):
  if i % 2 == 0:
    total += i
print(total)

Capstone: Word Frequency Counter

Combine Strings, Lists, Loops, and Dictionaries.

Task: Given a sentence: "cat bat cat rat bat cat"
1. Split the string into a list of words.
2. Loop through the list.
3. Use a dictionary to count how many times each word appears.
4. Final Output should look like: {'cat': 3, 'bat': 2, 'rat': 1}

Reveal Code Solution
text = "cat bat cat rat bat cat"
words = text.split() # Creates list of words
counts = {}

for word in words:
  if word in counts:
    counts[word] += 1
  else:
    counts[word] = 1

print(counts)
