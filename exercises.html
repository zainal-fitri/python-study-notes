1. Variables & Syntax

Ex 1: Variable Swap

Create two variables, a = 5 and b = 10. Swap their values so a becomes 10 and b becomes 5, then print them.

Show Solution
a = 5
b = 10

# The Pythonic way to swap
a, b = b, a

print(a)
print(b)

Ex 2: User Greeting

Ask the user for their name and their age. Print a sentence saying "Hello [Name], you will be [Age+1] next year."

Show Solution
name = input("Name: ")
age = int(input("Age: "))

# Convert to string to concatenate or use f-string
print("Hello " + name + ", you will be " + str(age + 1))

2. Data Structures

Ex 3: List Manipulation

Create a list of 3 fruits. Add a 4th fruit. Remove the 2nd fruit. Print the final list length.

Show Solution
fruits = ["Apple", "Banana", "Cherry"]
fruits.append("Date")
fruits.pop(1) # Removes index 1 (Banana)
print(len(fruits))

Ex 4: Dictionary Lookup

Create a dictionary representing a car with keys: brand, model, year. Print the model.

}
print(car["model"])

3. Control Structures & Loops

Ex 5: Even or Odd?

Ask the user for a number. Use an if/else statement to check if the number is even or odd (hint: use Modulus %).

Show Solution
num = int(input("Enter number: "))

if num % 2 == 0:
    print("Even")
else:
    print("Odd")

Ex 6: Sum of List

Given a list nums = [10, 20, 30], write a for loop to calculate the total sum manually (without using sum()).

Show Solution
nums = [10, 20, 30]
total = 0

for n in nums:
    total += n

print("Sum is:", total)

Final Challenge: Simple Calculator

Combine everything! Write a program that takes two numbers and an operator (+, -, *, /) from the user, performs the math, and prints the result.

Reveal Solution Code
n1 = float(input("First number: "))
op = input("Operator (+, -, *, /): ")
n2 = float(input("Second number: "))

if op == "+":
    print(n1 + n2)
elif op == "-":
    print(n1 - n2)
elif op == "*":
    print(n1 * n2)
elif op == "/":
    if n2 != 0:
        print(n1 / n2)
    else:
        print("Cannot divide by zero")
else:
    print("Invalid operator")
