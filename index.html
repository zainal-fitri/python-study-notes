General Description

Python is a high-level, interpreted general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation.

Interpreted

Python executes code line-by-line. If an error occurs, it stops immediately, making it excellent for debugging and learning.

Dynamically Typed

You do not need to declare variable types (like int or string) explicitly. Python figures it out at runtime.

Versatile

Used for Web Development, Data Science, Artificial Intelligence, Automation, and Scripting.

Identifiers & Syntax

Naming Rules (Identifiers)

  • Can contain letters (a-z, A-Z), numbers (0-9), and underscores (_).
  • Must start with a letter or an underscore.
  • Cannot start with a number.
  • Cannot contain spaces or special symbols like @, $, %.
  • Case Sensitive: Age and age are different.
Valid: my_var, _count, var2
Invalid: 2var, my-var, my var

Keywords (Reserved Words)

Words that have special meaning in Python. You cannot use these as variable names.

Indentation & Structure

Python uses whitespace indentation to define blocks of code (loops, functions, if-statements) instead of curly braces {}. Standard practice is 4 spaces.

Comments

Use # for single-line comments.

# This is a comment
x = 5 # Comment at end of line
if 5 > 2:
....print("Inside the block")
....# This runs only if true
print("Outside the block")

Multi-line Statements

Using Parentheses (Preferred)

Using Backslash

total = 10 + \

Essential Built-in Functions

print(...)

Output

Displays text or variables to the screen (console).

print("Hello World")
# Output: Hello World

input(...)

Input

Pauses the program to get text input from the user. Returns a string.

name = input("Your name?")

type(...)

Inspection

Tells you the data type (class) of a value or variable.

print(type(10))
# Output: <class 'int'>

int() / str()

Conversion

Converts values from one type to another (Type Casting).

x = int("5") # Becomes 5
y = str(10) # Becomes "10"
